# targets

CTF-specific material that is not generic lab infrastructure. Challenge logic, flag placement notes, and player-facing hints that belong in-repo as training OSINT.

## Three-part box spec

Each SecretCon challenge box should expose:

1. **Deployment** — runbooks under `docs/runbooks/` and/or Packer READMEs
2. **Attack FAQ** — `attack-faq-walkthrough.md` (player / red team)
3. **Defend FAQ** — `defend-faq-walkthrough.md` (defender / blue team)

See [docs/conventions.md](../docs/conventions.md) for canonical naming and box indexes.

| Box | Attack | Defend | Notes in `targets/` |
|-----|--------|--------|---------------------|
| CysVuln | [attack-faq](../docs/cysvulnserver/attack-faq-walkthrough.md) | [defend-faq](../docs/cysvulnserver/defend-faq-walkthrough.md) | — |
| EWS | [attack-faq](../docs/ews/attack-faq-walkthrough.md) | [defend-faq](../docs/ews/defend-faq-walkthrough.md) | `ews-win11/`, PCAP forensics |
| AS-REP | [attack-faq](../docs/asrep/attack-faq-walkthrough.md) | [defend-faq](../docs/asrep/defend-faq-walkthrough.md) | — |

## Contents

- `ews-win11/` — Win10 EWS kill chain notes (`flag-notes.md`) and related target config
- `ews-vnc-pcap-forensics/` — standalone analyst-track challenge
  ([`README.md`](ews-vnc-pcap-forensics/README.md)) bundling a real
  41-attempt TightVNC brute-force capture against the EWS VM. Players
  recover the foothold password from the chal/resp pair offline; the
  flag is `FELDTECH_VNC`. Pcap was generated by
  [`scripts/observability/vnc-public-attack.sh`](../scripts/observability/vnc-public-attack.sh)
  and validated by
  [`scripts/validate/validate-vnc-public-attack.sh`](../scripts/validate/validate-vnc-public-attack.sh)
  (5/5 PASS). Instructor walkthrough in
  [`ews-vnc-pcap-forensics/solution.md`](ews-vnc-pcap-forensics/solution.md);
  **do not ship `solution.md` to players.**

Do not move generic Packer or bootstrap code here; keep IaC under `infrastructure/` and `provisioning/`.

Intentional footholds documented in [docs/architecture.md](../docs/architecture.md) are not bugs.
